# LinkedIn Image Generator

Given a topic, key points, and optional on-image text, produce a finished
1080x1080 LinkedIn graphic (or a multi-slide carousel) in the NEA brand system:
- Colors: indigo {accent} on near-black {bg}, near-white text
- Fonts: Fraunces (headings), Hanken Grotesk (body)
- Include the NEA logo mark and a consistent footer
- Editorial, minimal, one clear focal message per slide

Do not write the caption — this skill only produces the image(s).
