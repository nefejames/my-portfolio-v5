# Client wordmark logos

Drop each brand's **wordmark** logo (the version that includes the company
name, not just the icon) here. It is picked up automatically — no code change.

## Naming

Use the client slug as the filename: `<slug>.svg` (or `.png` / `.webp`).

| Brand | File |
|---|---|
| LogRocket | `logrocket.svg` |
| Smashing Magazine | `smashing-magazine.svg` |
| Prismic | `prismic.svg` |
| AltexSoft | `altexsoft.svg` |
| Strapi | `strapi.svg` |
| ButterCMS | `buttercms.svg` |
| Dojah | `dojah.svg` |
| LoginRadius | `loginradius.svg` |
| Bird Eats Bug | `bird-eats-bug.svg` |

The portfolio filter pills use the same files, keyed by each article's
`clientSlug` (currently `altexsoft.svg` and `prismic.svg`).

## Dark background

The site is dark, so use logo versions that read on a dark background — a
**white / light or monochrome** wordmark is ideal. Standard dark-on-transparent
logos will be hard to see. SVG preferred (sharp at any size); transparent PNG is
fine. Until a file exists for a brand, the UI falls back to the brand name in
text.
